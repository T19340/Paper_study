# Paper Study — 채권 논문 정독 노트

수요기반 자산가격결정(demand-based asset pricing)과 국채시장을 다룬 학술 논문을, 한 편씩
원문째 깊이 정독한 **HTML 학습 노트** 모음입니다. 각 노트는 학부 기초 통계를 이수한 독자를
기준으로, 막히는 곳마다 사전지식을 그 자리에서 풀어 설명합니다.

각 노트의 구성:

- **원문 3층 인용** — 논문의 영어 원문을 그대로 인용하고, 한국어로 옮긴 뒤, 그 위에 해설을 답니다.
- **수식 풀이** — 모든 기호를 그 자리에서 말풀이하고, 중간 단계를 생략하지 않습니다(MathJax, 클릭하면 LaTeX 복사).
- **항해형 셸** — 좌측 본문 + 우측 sticky 목차·공식 패널, 스크롤 연동(scroll-spy), CAD 양식 도표.

## 보는 법

각 `.html` 파일을 내려받아 브라우저로 열면 됩니다(별도 설치 불필요, 인터넷 연결 시 수식이 렌더링됩니다).
GitHub 파일 화면에서는 소스가 보이므로, **Download raw** 후 브라우저로 여는 것을 권합니다.

## 출처·정직성

- 이 저장소에는 **정독 노트(HTML)만** 담겨 있으며, 원문 논문 PDF는 포함하지 않습니다.
- 모든 인용은 각 논문 원문에서 직접 따왔고, 수치는 본문·표·그림에서 인용하며 출처를 노트 안에 밝혔습니다.
- 학습·교육 목적의 해설 자료이며, 투자·정책 자문이 아닙니다.
